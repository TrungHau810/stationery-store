# Build a Stationery Store E-Commerce Website - Open Stationery Store
An online stationery e-commerce platform designed to provide customers with an easy, fast, and convenient shopping experience for office supplies, school materials, and help the owner manage products, categories, orders, and customers through an admin dashboard.

## Project Overview
The Stationery Store Website is a modern e-commerce platform designed for buying and selling stationery products online.  
This project aims to provide users with a convenient shopping experience, including product browsing, cart management, secure checkout, and order tracking.

## Objectives
- Build a responsive and user-friendly stationery shopping website.
- Support customer registration, login, and profile management.
- Allow users to browse, search, and filter stationery products.
- Implement shopping cart and order management features.
- Provide admin dashboard for managing products, categories, orders, and revenue.

## User roles
- Customer
- Staff
- Manager
- Admin (Owner)

## Key Features

### User Features
- User registration & login
- Browse stationery products by category
- Search and filter products
- Product details page
- Add to cart / remove from cart
- Place orders online
- Loyalty Points
- Order history tracking
- User profile management

### Admin Features
- Manage products (Create, Update, Delete)
- Manage categories
- Manage customer orders
- Sales statistics dashboard
- User management
